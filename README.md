# Project Setup Guide

## Requirements

Please install the following before running the project locally:

- Node.js (Recommended: Latest LTS version)
- npm (included with Node.js)

Download Node.js:

https://nodejs.org

Verify installation:

```bash
node -v
npm -v
```

---

# Install Dependencies

Open terminal inside the project folder and run:

```bash
npm install
```

---

# Build Project

Generate the production build (`dist` folder):

```bash
npm run build
```

---

# Run Project Locally

Start the local server:

```bash
npm start
```

---

# Open Application

After the server starts, open:

```txt
http://localhost:4173
```
